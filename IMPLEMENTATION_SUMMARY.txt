IMPLEMENTATION OVERVIEW
================================================================================

The Agent class has been enhanced with Hermes-style sub-agent management:

1. DEPTH TRACKING
   - Added _delegate_depth: int (0 = top-level)
   - Added _max_delegate_depth: int (default 5, configurable)
   - Prevents infinite recursion with depth limits
   - Hierarchical delegation: depth 0 → 1 → 2 → 3 → 4 → 5 (max)

2. ISOLATION
   - Each sub-agent gets its own message history
   - Separate compression state, execution plans, events
   - Shared: working_dir, tools, LLM config, parent context
   - Parent-child relationship tracked via parent_id field

3. PARALLEL EXECUTION
   - New delegate_parallel() method for concurrent tasks
   - Uses ThreadPoolExecutor with max 4 workers (safety limit)
   - Each task runs in isolated Agent instance
   - Returns list of results with status, duration, error tracking

4. CANCELLATION
   - New cancel_children() method signals all active children
   - Uses threading.Event for interrupt handling
   - Children check event during execution
   - Graceful shutdown without forceful termination
