# 安全审计模版 (Security Audit Template)

**tags:** security, vulnerability-assessment, compliance, penetration-testing
**roles:** Security Officer, Penetration Tester, Security Architect
**category:** Security & Compliance

## 目标 (Objective)

对系统进行全面安全评估，识别漏洞并制定修复和合规计划。

## OWASP Top 10 核查清单 (OWASP Top 10 Checklist)

### A01:2021 - 违反访问控制 (Broken Access Control)

**风险等级**: 极高

```
检查项目:

[ ] 用户权限验证
  - 每个操作都检查用户权限
  - 禁止直接对象引用（IDOR）检查
  - 用户 ID 伪造测试
  - 跨用户数据访问测试

[ ] 水平权限提升
  - 用户 A 能否访问用户 B 数据
  - 修改 URL 参数绕过权限
  - 修改请求体绕过权限

[ ] 垂直权限提升
  - 普通用户能否执行管理员操作
  - 后台 URL 直接访问
  - 权限检查被移除

[ ] 会话管理
  - 会话 ID 是否随机且不可预测
  - 会话是否正确过期
  - 注销后会话是否失效
  - 并发会话控制
```

**检查状态**: [ ] 通过 / [ ] 失败 / [ ] 部分通过

### A02:2021 - 密码学失败 (Cryptographic Failures)

**风险等级**: 极高

```
检查项目:

[ ] 传输中数据加密
  - HTTP vs HTTPS
  - TLS 版本 (1.2+)
  - 强加密套件
  - 证书验证

[ ] 存储数据加密
  - 敏感数据加密
  - 密钥管理 (HSM/KMS)
  - 密钥轮换计划
  - 密钥备份和恢复

[ ] 密码存储
  - 使用强哈希（Argon2/bcrypt）
  - 适当的 salt 大小
  - 工作因子/迭代次数
  - 不使用 MD5/SHA-1

[ ] 随机数生成
  - 密码学安全的随机数
  - PRNG vs CSPRNG
  - 种子来源可靠性
```

**检查状态**: [ ] 通过 / [ ] 失败 / [ ] 部分通过

### A03:2021 - 注入漏洞 (Injection)

**风险等级**: 极高

```
检查项目:

[ ] SQL 注入
  - 所有数据库查询参数化
  - 预编译语句使用
  - 动态 SQL 构建避免
  - ORM 配置检查

[ ] NoSQL 注入
  - 输入验证
  - 参数绑定（不适用可避免）
  - 查询构建审查

[ ] 命令注入
  - 避免 shell 执行
  - 参数化执行
  - 输入白名单

[ ] LDAP 注入
  - LDAP 查询参数化
  - 输入转义

[ ] XPath 注入
  - XPath 表达式参数化
```

**检查状态**: [ ] 通过 / [ ] 失败 / [ ] 部分通过

### A04:2021 - 不安全设计 (Insecure Design)

**风险等级**: 高

```
检查项目:

[ ] 设计审查
  - 安全要求文档化
  - 威胁建模完成
  - 安全设计模式应用

[ ] 身份验证流程

[ ] 密码策略

  - 定期更换

[ ] 业务逻辑验证
  - 转账金额验证
  - 库存充足性
  - 订单价格验证
```

**检查状态**: [ ] 通过 / [ ] 失败 / [ ] 部分通过

### A05:2021 - 安全配置错误 (Security Misconfiguration)

**风险等级**: 高

```
检查项目:

[ ] 服务器配置
  - 移除不必要服务
  - 默认密码变更
  - 不必要端口关闭
  - 安全头配置 (HSTS/CSP/X-Frame-Options)

[ ] 依赖项管理
  - 依赖项最新版本
  - 已知漏洞检查 (CVE)
  - 自动更新配置
  - 供应链风险评估

[ ] 云配置
  - S3 桶不公开
  - IAM 权限最小化
  - 加密启用
  - 日志记录启用
  - VPC 配置安全

[ ] 容器配置
  - 镜像扫描
  - 多层安全检查
  - 运行时安全
  - 网络策略

[ ] 错误处理
  - 不暴露系统信息
  - 通用错误消息
  - 安全日志记录
```

**检查状态**: [ ] 通过 / [ ] 失败 / [ ] 部分通过

### A07:2021 - 身份验证失败 (Authentication Failures)

**风险等级**: 高

```
检查项目:

[ ] 密码破解防护
  - 速率限制实施
  - 账户锁定机制
  - CAPTCHA 集成
  - 登录尝试日志

[ ] 会话固定
  - 登录后会话 ID 重新生成
  - 会话 cookie 安全标志
  - HTTPOnly 标志
  - Secure 标志 (HTTPS)
  - SameSite 属性

[ ] 凭证暴露
  - 密码不存储在内存中
  - 密码不记录
  - 密码不在 URL 中
  - 安全传输

[ ] 恢复机制
  - 密码重置令牌有效期短
  - 重置令牌一次性使用
  - 没有发送当前密码
  - OTP 一次性使用
```

**检查状态**: [ ] 通过 / [ ] 失败 / [ ] 部分通过
