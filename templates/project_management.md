# 项目管理模版 (Project Management Template)

**tags:** project-management, agile, sprint, stakeholder-management
**roles:** Project Manager, Scrum Master, Product Owner
**category:** Project Management

## 目标 (Objective)

规划、协调和监控项目执行，确保按时、按预算、按质量交付。

## 风险管理 (Risk Management)

### 风险识别 (Risk Identification)

```
风险 ID: [ID]
风险: [描述]
触发器: [可能导致的事件]
影响: [如果发生会怎样]
概率: [ ] 低 / [ ] 中 / [ ] 高
影响大小: [ ] 低 / [ ] 中 / [ ] 高
风险等级: [ ] 低 / [ ] 中 / [ ] 高 / [ ] 极高

优先级排序:
风险等级 = 概率 × 影响
```

### 风险评估矩阵 (Risk Assessment Matrix)

```
                  低影响    中影响    高影响
低概率        低风险    低风险    中风险
中概率        低风险    中风险    高风险
高概率        中风险    高风险    极高风险

应对策略:
极高: 立即采取行动，可能需要项目变更
高:   制定具体缓解计划，监控
中:   识别缓解计划，定期审查
低:   记录，继续监控
```
